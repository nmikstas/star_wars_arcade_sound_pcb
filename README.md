# star_wars_arcade_sound_pcb
Reproduction of the Star Wars arcade sound PCB. The PCB is complete but not yet verified.

# Folder Structure
* BOM - Bill of materials for the sound PCB
* Gerbers - The gerber files for the sound PCB
* images - The images used for the sound PCB
* pcb - The Kicad design files for the sound PCB

# PCB Specifications
* Layers: 2
* Dimensions: 293mm X 227mm
* Different PCB Design: 1
* PCB Thickness: 1.6mm
* Surface Finish: HASL
* Castellated Holes: No
* Copper Weight: 1oz.
